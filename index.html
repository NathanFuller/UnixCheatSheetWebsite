The Unix Cheat Sheet

Welcome to my website! Here, you will learn some interesting things you can do on the command line.

Some useful commands

Command Description Usage Example
ls "List"; shows a list of files and directories in your working directory. 
username:~/workspace $ ls
2048.py/  B-2.jpeg  UnixCheatSheet/  sandbox/
cd "Change Directory"; allows you to move to a different directory. username:~/workspace $ cd UnixCheatSheet/
man Opens the "manual" for the command you tell it username:~/workspace $ man wget
wget Give this command a URL and it downloads the file found there username:~/workspace $ wget http://www.airforce-technology.com/uploads/newsarticle/1000388/images/201017/large/3-f-22-raptor.jpg
